# PRUEBAESTUDIANTE
# README
Welcome. This is a simple computer program that helps you remember all the students in your school. It automatically saves all the information in a special file so you never lose your list! What can you do with it?
The following available options will appear on the screen or terminal:

1. Create Student
2. List Students
3. Search Student
4. Update Student List
5. Delete Student
6. Exit
Select

CREATE STUDENT: Enter the name, age, and grade of a new student, indicating whether they are active or inactive.

LIST STUDENTS: View the complete list of all the students you have saved.

SEARCH:
Checks the data for a specific student.

UPDATE LIST:
If a student has changed their age or name, or made a mistake during registration, select this option to correct or update this information.

DELETE STUDENT: Removes a student from the register if they leave the school.

EXIT: Finally, if you no longer need to do anything, click Exit. All data will be saved, and you will exit the program.
